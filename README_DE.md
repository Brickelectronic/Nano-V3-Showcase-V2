# Showcase Shield V2
LEGO® kompatibles Shield für Arduino Nano V3
<br><br>
Das Shield wurde für LEGO User Groups entwickelt, um MOCs in Vitrinen mit Beleuchtungen auszustatten und diese von außen mit Näherungssensoren zu steuern.
